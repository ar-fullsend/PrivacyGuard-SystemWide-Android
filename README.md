# PrivacyGuard-SystemWide-Android

**Your entire screen. Protected. Automatically.** (Android version)

Real-time, camera-driven, system-wide privacy shield using **CameraX + ML Kit**.

## Features
- Full-screen privacy overlay when second face detected
- On-device ML face detection (ML Kit)
- Power-efficient camera pipeline
- Click-through overlay (user can still interact)
- Hotkey / quick toggle support

## Tech Stack
- Kotlin + Jetpack Compose
- CameraX
- ML Kit Face Detection
- Android 10+ (API 29+)

## Quick Start
1. Clone the repo
2. Open in Android Studio
3. Build & Run
4. Grant camera permission
5. The app will monitor and protect your screen
